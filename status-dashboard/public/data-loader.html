  • Browse file API: Use the File API to verify uploaded files

  • Download files: Access individual files at /api/data/filename.json
  • Add new files: Copy files directly to the data folder or upload via ZIP

Prepare for screening

The "Prepare for screening" button processes all files in your data folder and loads them into a search index for screening.

This process:

  • Scans all .json files in the data folder
  • Validates the file format and structure
  • Loads valid files into the dsfp-screening-index Elasticsearch index
  • Creates tracking records for file management
  • Makes files available for screening analysis

Use this button after uploading new files or when you want to refresh the screening index with the latest data.
