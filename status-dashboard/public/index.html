About DSFP in a Box

Digital Sample Freezing Platform - in a box!

This is a containerized prototype of DSFP services, designed to run entirely on a localized environment without requiring internet connectivity for analysis operations.

What DSFP Does

DSFP performs advanced non-targeted screening of samples against the NORMAN Substance Database compounds to:

  • Identify substances: Match spectral patterns to known compounds in the database
  • Screen samples: Process LC-MS/MS data files to detect substances of interest
  • Analyze results: Generate comprehensive reports with spectral similarity scores
  • Track progress: Maintain a complete audit trail of all screening operations

How It Works

  1. Data Storage: Upload DSFP data (JSON format)
  2. Compound Database: Pre-loaded with 95,000+ compounds and their spectral characteristics
  3. Screening: Select samples and run automated screening against the compound database
  4. Results: Review matches with spectral similarity scores, retention times, and detailed metadata

System Components

  • Elasticsearch: High-performance database for compounds and sample data
  • Screening Service: Python-based API for running screening algorithms
  • Status Dashboard: Web interface for monitoring and management
  • DuckDB: Local database for tracking screening progress and results
